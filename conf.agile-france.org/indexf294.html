Un template est disponible pour les orateurs

Posted on 25/04/2012 by Eric Le Merdy
Tweet

Les orateurs ont été sélectionnés. On peut les retrouver sur le programme 2012.

Format LibreOffice

Format Microsoft Powerpoint

Format Microsoft Office Open XML

Format Apple Keynote
